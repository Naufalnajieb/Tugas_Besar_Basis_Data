# Tugas_Besar_Basis_Data

Berikut ini adalah website yang saya buat sebagai bagian dari proyek Tugas Besar dalam Mata Kuliah Sistem Basis Data. Website ini berfungsi sebagai platform sistem informasi penyewaan lapangan futsal Moriz Futsal yang dikembangkan menggunakan PHP Native, serta diintegrasikan dengan database MySQL untuk pengelolaan data pelanggan, lapangan futsal, transaksi sewa, dan pembayaran. Sistem ini dirancang untuk memudahkan pelanggan dalam melakukan booking lapangan futsal secara online dengan opsi item ekstra, penggunaan voucher diskon, dan pembayaran yang efisien.

## Penjelasan Desain Program
1. User melakukan registrasi/login -> Pelanggan dan Pegawai (Admin) dapat mendaftar dan masuk ke sistem
2. Pelanggan memilih Lapangan -> Pelanggan memilih lapangan futsal yang tersedia dengan harga dan fasilitas yang sesuai
3.  Pelanggan membuat Transaksi Sewa -> Membuat booking dengan menentukan tanggal sewa dan jam awal & akhir penggunaan
4. Pelanggan dapat menambahkan Item Ekstra -> Opsi untuk menambahkan peralatan atau layanan tambahan (cones, bola, dll)
5. Penggunaan Voucher Diskon -> Pelanggan dapat menggunakan kode voucher untuk mendapatkan potongan harga
6. Pelanggan melakukan Pembayaran -> Membayar dengan berbagai metode pembayaran dan memberikan bukti pembayaran
7. Admin mengelola Status Pembayaran -> Pegawai/Admin melakukan verifikasi dan update status pembayaran

## Keunggulan Web
1. User & Admin Management
2. Status Management Pembayaran
3. Sistem Voucher & Diskon
4. Inventory Management
5. Tracking Bukti Pembayaran
6. Audit Trail Lengkap
